# DLA Simulation

A simple diffusion-limited aggregation (DLA) simulation built with Python, NumPy, and Matplotlib.

## About this project

This project simulates particles moving randomly in a grid and sticking to each other forming patterns.

This project uses:
- simulation programming
- array-based computation
- visualization with Matplotlib

## Features

- Random moving particles
- Grid-based simulation
- Emergent patterns

## Tech Stack

- Python
- NumPy
- Matplotlib

Clone the repository:

```bash

cd DLA-Simulation/

python main.py
```
